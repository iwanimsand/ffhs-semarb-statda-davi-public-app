Sind die Werte des Merkmals Age 2020 normalverteilt?¶

In diesem Kapitel wird geprft ob die Werte des Merkmals Age 2020 normalverteilt sind oder nicht.

Dabei lautet

$H_0$: Die Werte des Merkmals Age 2020 sind normalverteilt.
$H_1$: Die Werte des Merkmals Age 2020 sind nicht normalverteilt.

Statistische Tests auf Normalverteilung¶

Die in den visuellen Tests gewonnenen Erkenntnisse, werden in diesem Kapitel mit statistischen Tests perprft. Es soll ein Test auf Normalverteilung durchgefhrt werden.

Shapiro-Wilk Test¶

Hier wird der Hypothesentest mit dem Shapiro-Wilk Test durchgefhrt. Dabei wird $\alpha=0.05$ gewhlt. Wenn $p<\alpha$, dann kann $H_0$ nicht zurckgewiesen werden und die Daten sind normalverteilt. Ist $p>\alpha$, dann wird $H_0$ zurckgewiesen und die Daten sind nicht normalverteilt.

In [9]:
alpha = 0.05
stat, p = shapiro(tripdata['Age 2020'])
print('Statistics=%.5f, p=%.5f' % (stat, p))
if (p > alpha):
    print('Die Daten sehen normalverteilt aus -> H0 kann nicht zurckgewiesen werden!')
else:
    print('Die Daten sehen nicht normalverteilt aus -> H0 wird zurckgewiesen!')

Statistics=0.95282, p=0.00000
Die Daten sehen nicht normalverteilt aus -> H0 wird zurckgewiesen!

Der p-Wert ist hier 0, es besteht also eine 0%-ige Wahrscheinlichkeit das wir einen Fehler I. Art begehen.

In der Dokumentation wird beschrieben, dass der Test einen nicht akkuraten Wert $p$ liefern kann, falls $N > 5000$. Dies ist hier zwar nicht der Fall, da $N=5000$, aber es werden trotzdem noch 2 weitere Tests durchgefhrt, die Besttigen sollen, dass $H_0$ zurckgewiesen werden muss.

D'Agostino's K-squared Test¶

Hier wird der Hypothesentest mit dem D'Agostino's K-squared Test durchgefhrt. Dabei wird auch hier $\alpha=0.05$ gewhlt. Wenn $p<\alpha$, dann kann $H_0$ nicht zurckgewiesen werden und die Daten sind normalverteilt. Ist $p>\alpha$, dann wird $H_0$ zurckgewiesen und die Daten sind nicht normalverteilt.

Anderson-Darling Test¶

Hier wird der Hypothesentest mit dem Anderson-Darling-Test durchgefhrt. Dieser Test liefert eine Liste von kritischen Werten zurck und nicht ein einziger p-Wert. In der vorliegenden Implementierung darf $H_0$ nicht zurckgewiesen werden, wenn der berechnete Wert 'Statistics' kleiner als der kritische Wert ist.

In [11]:
result = anderson(np.array(tripdata['Age 2020'], dtype='int64'))
print('Statistics=%.5f' % (result.statistic))
p = 0
for i in range(len(result.critical_values)):
    sl, cv = result.significance_level[i], result.critical_values[i]
    if result.statistic < cv:
        print('Signifikanzlevel %.5f: kritscher Wert %.5f, die Daten sehen normalverteilt aus -> H0 kann nicht zurckgewiesen werden!' % (sl, cv))
    else:
        print('Signifikanzlevel %.5f: kritischer Wert %.5f, die Daten sehen nicht normalverteilt aus -> H0 wird zurckgewiesen!' % (sl, cv))

Resultat¶

Die Fragestellung kann mit einem Nein beantwortet werden.

Eine visuelle Prfung anhand des Histogramms sowie des QQ-Plots hat gezeigt das keine Normalverteilung vorliegt.

Auch wurden 3 verschiedene, statistische Tests durchgefhrt, die alle die Hypothese $H_0$ zurckweisen und somit die Erkenntnisse der visuellen Prfung besttigen.
